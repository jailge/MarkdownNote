# 删除事务


## URL
```
http://10.10.6.72/api/v1/affair/datadelete
```


## 参数

|  名称   | 说明    |  类型 |
| --------- | -------- | -----: |
| uuids  | 事务uuid列表  | UUID |



## 删除示例

```json
{
	"uuids": ["55384128-adf2-11e9-8baa-8c859028b48d", "......."]
}
```

## 成功结果

```json
{
    "del": [
        "55384128-adf2-11e9-8baa-8c859028b48d"
    ],
    "total": 1
}
```
