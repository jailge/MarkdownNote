# 搜索事务


## URL
```
http://10.10.6.72/api/v1/affair/datasearch
```

## 调用方法

```
POST
```

## 参数

|  名称   | 说明    |  类型 |
| --------- | -------- | -----: |
| 属性  | 属性名称  | 字符串 |
|   值   |   属性值  |        |
|   比较符 |   查询比较符 |  equal/like/begin with/end with/greater than or equal to/less than or equal to/greater than/less than/between/sort       |



## 查询示例
1. 查询所有事务
```json
{
	"事务类别":"一般事务",
	"条件":[
		{"属性":"uuids", "值":"uuids", "比较符":"equal"}
	]
}
```

2. 多条件查询
```json
{
	"事务类别":"一般事务",
	"条件":[
		{"属性":"是否完成", "值":"0", "比较符":"equal"},
		{"属性":"处理人", "值":"丁", "比较符":"begin with"}
	]
}
```
